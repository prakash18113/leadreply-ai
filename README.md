# LeadReply Cloud

Mobile-adaptive SaaS starter built with Next.js and shadcn/ui.

## What this project includes

- Marketing landing page with product positioning, trust logos, and conversion CTAs.
- In-app SaaS dashboard with views for messages, analytics, settings, and billing.
- Monetization-ready UX with pricing tiers and billing history UI.
- Security and cloud-focused product messaging (RBAC/API key controls, autoscaling narrative).

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
